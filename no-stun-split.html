Local Network Demo

This demo does not have a STUN server configured. It is intended to work on a local network. It was developed with opening the initiator and receiver in different tabs on the same browser. You can also open the initiator and receiver on different browsers or devices instead of tabs. For different devices firewalls may cause issues blocking UDP ports used for the negotiation. Try reversing the initator and receiver roles if you run into this issue.
